This help text can be sent to you via private message by typing !help in a chat room where lujan-bot is present.

Here's a list of each command you can type, followed by a small description of how to use it:
  
!help - Displays this help text.
!ping - Pongs you back.
!gh commit <repo> <id> - Displays info for a commit in one of Eric's GitHub repositories
